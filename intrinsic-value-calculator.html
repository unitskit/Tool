Formula Reference
DCF Valuation

Present Value = FCF / (1 + WACC)^n

DDM Valuation

Value = DPS / (r - g)

Cost of Equity

r = Rf + β(Rm - Rf)

Variables:
  • FCF = Free Cash Flow
  • WACC = Weighted Average Cost of Capital
  • DPS = Dividend per Share
  • r = Required Return
  • g = Growth Rate
  • β = Beta
  • Rf = Risk-Free Rate
  • Rm = Market Return
